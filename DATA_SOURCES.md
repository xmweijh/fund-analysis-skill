## 📊 总体架构

```
┌─────────────────────────────────────────────────────┐

        ┌──────────▼──────────┐

        └──────────┬──────────┘
                   │
    ┌──────────────┼──────────────┐
    │              │              │
    ▼              ▼              ▼
┌────────────┐ ┌─────────────┐ ┌──────────┐
│ 用户持仓   │ │ 默认基金    │ │ 降级方案 │
│ portfolio │ │ 集合 (10只) │ │ (5只)    │
│ .json     │ │             │ │          │
└────────────┘ └─────────────┘ └──────────┘
    │              │              │
    └──────────────┼──────────────┘
                   │
```

## 📌 推荐基金列表来源 (优先级)

### 优先级 1️⃣：用户持仓列表 (portfolio.json)

#### 来源路径
- **文件位置**: `data/portfolio.json`
- **读取方式**: PortfolioManager 类
- **更新时机**: 用户执行 `add` 或 `remove` 命令时

#### 数据内容
```json
[
  {
    "fund_code": "008269",
    "fund_name": "大成睿享混合A",
    "shares": null,
    "cost_nav": null,
    "cost_amount": null,
    "note": "",
    "added_at": "2026-03-07 15:00:00",
    "updated_at": "2026-03-07 15:00:00"
  },
  // ... 更多基金
]
```

#### 优势
- ✅ 用户自己管理，完全可控
- ✅ 推荐与用户风格一致
- ✅ 私密性强，本地存储
- ✅ 支持 add/remove 动态管理

#### 使用场景
- 用户想找相似风格的其他基金
- 用户想对比自己持仓和推荐
- 长期用户积累的基金历史

## 🔍 每只基金的详细分析数据源

### 通过蛋卷 API 获取的数据

#### 1. 基础信息 (DanjuanDataFetcher.fetch_basic_info)
```
- 基金代码 (fund_code)
- 基金名称 (fund_name)
- 基金类型 (fund_type)
- 成立日期 (setup_date)
- 基金公司 (fund_company)
- 基金规模 (fund_scale)
```

**数据来源**: 蛋卷基金 API  
**更新频率**: 实时  
**可靠性**: ⭐⭐⭐⭐⭐

---

#### 2. 实时行情 (DanjuanDataFetcher.fetch_realtime_quote)
```
- 最新净值 (price)
- 日涨跌幅 (change_percent)
- 日期时间 (update_time)
```

**数据来源**: 蛋卷基金 API  
**更新频率**: 每交易日  
**可靠性**: ⭐⭐⭐⭐⭐

---

#### 3. 净值历史 (DanjuanDataFetcher.fetch_nav_history)
```
- 365天的日净值数据
- 日期、净值、累计净值
```

**数据来源**: 蛋卷基金 API  
**覆盖范围**: 最近365天  
**用途**: 技术面分析（均线、趋势）  
**更新频率**: 每日  
**可靠性**: ⭐⭐⭐⭐⭐

---

#### 4. 持仓数据 (DanjuanDataFetcher.fetch_holdings)
```
- 前十大持仓股票
- 持仓占比
- 行业分布
- 持仓集中度
```

**数据来源**: 蛋卷基金 API  
**覆盖范围**: 最近1-3个月（定期公告）  
**更新频率**: 定期（可能有延迟）  
**可靠性**: ⭐⭐⭐⭐

---

#### 5. 基金经理信息 (DanjuanDataFetcher.fetch_manager_info)
```
- 基金经理名称
- 管理年限
- 从业年限
- 管理历史业绩
```

**数据来源**: 蛋卷基金 API  
**更新频率**: 实时  
**可靠性**: ⭐⭐⭐⭐⭐

---

#### 6. 业绩数据 (DanjuanDataFetcher.fetch_performance)

```
- 近1月/3月/6月/1年收益率
```

**数据来源**: 蛋卷基金 API  
**覆盖范围**: 全历史  
**更新频率**: 每日  
**可靠性**: ⭐⭐⭐⭐⭐

## 🔧 数据获取的错误处理

### 异常场景 1：API 不可用
```python
# 特定数据获取失败，不影响其他数据
if not nav_history:
    technical = None  # 跳过技术分析
    logger.warning("净值历史获取失败，跳过技术面分析")
```

### 异常场景 2：部分数据缺失
```python
# 使用可用数据继续评分
fund_input = FundRecommendationInput(
    technical=technical,  # 可能为 None
    holding=holding,      # 可能为 None
    manager=manager,      # 可能为 None
    # ... 评分器会处理 None 值
)
```

### 异常场景 3：完全无数据
```python
# 返回默认基金列表（降级方案）
if not fund_scores:
    logger.warning("分析失败，使用默认基金列表")
    return default_funds[:5]
```

## 📚 数据来源参考

### 蛋卷基金 API
- **库**: `pysnowball` (Python Snowball)
- **接口**: 蛋卷基金 (Danjuan.com)
- **文档**: [pysnowball GitHub](https://github.com/ce007/pysnowball)
- **数据范围**: 全市场基金
- **可靠性**: 业界常用，数据准确

### 东方财富 API
- **库**: 自定义爬虫
- **接口**: 东方财富网 (East.com)
- **数据**: 基金公告、舆情信息
- **可靠性**: 官方渠道，数据可信

### 本地存储
- **格式**: JSON
- **路径**: `data/portfolio.json`
- **管理**: 用户自主添加/删除
- **可靠性**: 本地数据，100% 准确

## 💡 使用建议

### 何时更新推荐

- **日常推荐**: 每天 1-2 次
- **重要决策**: 市场大变动后立即更新
- **新用户**: 首次添加持仓后立即查看推荐

### 数据刷新策略

```
推荐报告
    ↓
每 24 小时过期
    ↓
用户手动刷新
    ↓
获取最新数据
    ↓
重新评分排序
    ↓
生成新报告
```

### 推荐结合使用

1. **持仓+推荐**: 对比自己持仓和系统推荐
2. **多维分析**: 查看推荐基金的详细分析报告
3. **定期更新**: 定期获取新推荐，避免信息过时
4. **风险管理**: 根据自身风险承受能力选择
